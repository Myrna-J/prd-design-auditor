# 核心战术武器库 (Core Skills 2.0 Library)

此目录存放经过 **Skills 2.0** 审计与重构的专家级指令集。这是配置中心的单点真相 (SSOT)。

## 💎 特色规格
所有本库内的技能均强制要求：
1. **🔬 Symptoms & Triggers**: 精确定义 AI 应该在什么场景下激活。
2. **🛠️ 7-Step Workflow**: 标准化的专家级执行序列。
3. **✅ Action Checklist**: 物理级合规检查点。
4. **🧠 Rationalization Table**: 针对性对抗 AI 偷懒行为的逻辑锚点。
