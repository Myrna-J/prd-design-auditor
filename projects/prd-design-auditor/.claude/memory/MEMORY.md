# 项目主记忆 (Main Memory)

## 项目背景
本项目旨在为项目经理（PM）提供一套基于 AI 的辅助工具，专门用于审计供应商交付的「不动产经营管理平台」PRD 文档与 UI 设计图，解决业务需求模糊与开发文档质量差的矛盾。

## 核心约定
1. **纯中文协议**: 除非是特定的技术术语（如 API, JWT, TDD），所有文档标题、内容描述、复盘记录必须使用简体中文。
2. **业务锚定**: 所有逻辑必须基于「不动产经营」领域知识。
3. **分层审计**: 遵循“需求标准化 -> 文档穿透审计 -> 经营效率校验”的三层防御体系。

## 已知反模式 (Anti-patterns)
- **语言偏移**: 警惕 AI 自动回退到英文标题或繁体中文，必须强制简体中文输出。
- **通用化陷阱**: 避免给出“通用的 PRD 建议”，必须结合不动产的具体业务（如租金递增、公摊计算）。

## 关键决策
- **技术选型**: 初期采用 CLI 结构化报告辅助 PM，后端规则库基于 Markdown 维护。
- **技能绑定**: 强绑定 `product-strategy-context (v2.0)` 与 `ui-ux-design-audit (v2.0)` 的汉化适配版。
